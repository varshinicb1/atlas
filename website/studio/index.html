Atlas Studio

Visual knowledge graph explorer. Browse, search, and explore knowledge packages as interactive graphs.

Run Locally Features

Features

  • Interactive graph — React Flow-powered visualization with zoom, pan, and MiniMap
  • Registry browsing — Browse all 51 public packages directly from the Atlas registry
  • Local bundles — Load any compiled .atlas file from your filesystem
  • Node filtering — Search and filter nodes by name or ID
  • Detail panel — Click any node to see its connections, metadata, and description
  • Dagre layout — Automatic left-to-right graph layout

Run Locally

Atlas Studio requires the atlas CLI binary and runs on your machine:

# Build the CLI first
$ cargo build --release

# Start the studio
$ cd studio && npm install
$ npm run dev

# Open http://localhost:3000

Once running, you can:

  • Click Registry to browse and load packages from the public registry
  • Click Local to enter a path to a compiled .atlas file
  • Click any node to see its details and connections
  • Use the search bar to filter nodes
